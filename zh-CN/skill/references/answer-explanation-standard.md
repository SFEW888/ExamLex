# 全科详细答案解析标准

ExamLex 生成模拟卷、讲解练习、复盘错题或提供带教学说明的答案时，必须采用本标准。本标准适用于 `CET4`、`CET6`、`POSTGRADUATE_ENGLISH`、`TEM4`、`TEM8`，并覆盖 `vocabulary`、`listening`、`reading`、`translation`、`writing`、`cloze`、`language-knowledge`、`proofreading`、`dictation` 全部模块。

## 目标与证据边界

解析必须教会学习者看懂任务、找到答案、排除其他选项并自行核验。只有答案、只有一句结论，或用一段套话合并讲解多道题，都不算完整解析。

运营者提供的商业教辅答案页或带标注示例只能用于借鉴呈现结构，证据角色一律是 `R` 级教学方法参考，不是官方试卷、官方答案库、固定题库或新题的直接题源。不得复制其中的题目、文章、录音稿、范文、译文或解析文字。借鉴其排版方式不会提高任何来源的证据等级。

## 默认深度与辅助语言

所有学习者和所有完整模拟卷一律默认使用 `detailed` 详细档。不得因为学习者属于 `基础偏弱`、`中等基础`、`基础较好`、目标分数较高或考试类型不同而静默减配。学习者可以对某一次输出明确要求只看简明答案，但这不会改变系统保存的默认值。

使用学习者选定的辅助语言。面向默认中文学习者时，必须提供中文翻译和中文解析。基础较好的学习者仍应获得所有必填字段；可以压缩真正简单的词义说明，但证据、翻译、干扰项分析、错误诊断和迁移练习仍是必填项。

对 `基础偏弱` 学习者，在完整标准之上增加脚手架：

- 用通俗中文解释必要词汇和语法术语；
- 把长句与推理拆成更小的编号步骤；
- 明确画出题干、证据与答案之间的关系；
- 区分字面义与语境义；
- 最后给一个简短重做或迁移任务。

## 核心输出契约

### 客观题或选择题

每道题必须单独解析。学习者端可以使用自然的 Markdown 标题，但底层概念字段必须完整：

| 字段 | 必填内容 |
|---|---|
| `answer_key` | 题号和唯一、明确的答案。 |
| `tested_skill` | 题型与考查能力。 |
| `question_translation` | 题干或陈述的准确中文翻译；否定、情态、比较、数字、时间与范围不得丢失。 |
| `option_translation` | 翻译所有选项，包括全部错误选项；必须保留 only、mainly、may、all、except 等限制词。 |
| `evidence_anchor` | 支持答案的准确录音行/时间点、段落与句子、空格或语言规则；不得编造定位。 |
| `evidence_scope` | 明确答案来自同一句、相邻上下文、整段综合、全文结构，还是语法/用法规则。 |
| `evidence_translation` | 翻译决定性证据句或短证据片段，必要时还原代词指代和省略信息。 |
| `reasoning_steps` | 从题干到证据、推断和结论的编号推理步骤。 |
| `paraphrase_map` | 证据与答案之间的同义替换、改写、逻辑关系、指代链或音形对应。 |
| `key_language` | 学习者需要掌握的词汇、搭配、语法、语音或长难句知识。 |
| `distractor_analysis` | 逐项解释每个错误选项为什么错：与原文矛盾、未提及、范围偷换、概念偷换、过度推断、语法不成立或语境不合。 |
| `error_tag` | 该题能诊断学习错误时，选择最匹配的 ExamLex 错误标签。 |
| `learner_retry` | 一个可执行的重做动作：重新定位、重听片段、重拆长句、重译选项或完成平行微题。 |

不得只写“原文提到 B”。必须给出简短决定性证据、证据翻译、证据范围和推断链。证据引用应保持简短。如果复核后仍有两个选项同样成立，应修改题目，而不是强行指定唯一答案。

### 主观题

写作、翻译及其他开放式任务共用以下底层字段：

| 字段 | 必填内容 |
|---|---|
| `task_analysis` | 文体、读者、目的、话题、内容要点、立场或交际目标、限制与评分重点。 |
| `answer_plan` | 按顺序列出的段落提纲或分句方案。 |
| `reference_answer` | 原创、符合考试要求的参考答案；不得暗示只有这一种写法。 |
| `reference_translation` | 与参考答案逐句或逐段对齐的准确中文译文。 |
| `scoring_points` | 按评分维度说明得失分点，并明确分数只是训练估分而非官方评分。 |
| `language_notes` | 可用结构、词汇、搭配、衔接、时态、语态与语域说明。 |
| `acceptable_alternatives` | 同样应得分的准确表达、结构、论点或组织方式。 |
| `common_errors` | 常见遗漏、误译、语法问题、逻辑薄弱或语域不当及其修正方法。 |
| `learner_revision` | 一个具体的改写、自检或对比任务。 |

## 阅读与听力的共用材料包

完整详细答案册中的每篇阅读或每段听力材料必须：

1. 给出本次原创模拟实际使用的完整、合规文章或录音稿；
2. 按段提供通顺的中文全文翻译；
3. 提供聚焦复习的 `core_vocabulary`，包含语境义、常用搭配与关键长难句结构；
4. 为段落、句子或录音行编号，使 `evidence_anchor` 可以复核；
5. 在逐题解析前翻译每道题干和全部选项；
6. 标明 `evidence_scope`，翻译证据，说明同义替换或推断链，并逐项排除全部干扰项。

如果材料来自用户提供且受版权保护的试卷，不得超出用户授权范围大段重现。不能完整转载时，应依托用户已提供的材料讲解，只使用短引文或准确位置说明。

## 分模块规则

### 词汇 `vocabulary`

- 给出语境义、词性、词形变化、词族和中文义。
- 相关时补充发音或辨音、高频搭配和一个考试语境例句。
- 说明其他词义或近义词为何不适合当前语境。
- 将错误归入识义、拼写、听音识词或语境运用等标签。

### 听力 `listening`

- 原创模拟在限时作答后提供完整可读录音稿及逐段中文翻译，并为所有出题证据编号或标时间点。
- 每题先翻译题干和所有选项，再讲答案。
- 说明题型以及决定答案的信号词、说话人态度、话轮、数字、日期、转折或因果关系。
- 引用并翻译决定性录音句，明确答案来自单一话轮、相邻话轮、整段对话还是全文综合，再展示音形对应与同义替换链。
- 必要时解释连读、弱读、缩读、同音词、指代词和数字陷阱；所有干扰项都必须与录音逐项核对，包括“未提及”。
- 最后给出精确重听范围和重听目标，不能只写“再听一遍”。

### 阅读 `reading`

- 提供全文翻译和核心词汇表。
- 说明题型，翻译题干和所有选项，定位段落与句子，翻译证据，标注证据范围，列出同义替换，并拆解决定性长难句。
- 明确说明答案属于原句直接复现、局部上下文推断、整段综合还是全文结构推断。
- 每个错误选项单独排除，并指出其具体错误：逻辑颠倒、无依据细节、主语错误、时间偷换、范围扩大/缩小、绝对化或“看似合理但原文未说”。

### 四级专项规则 `CET4-specific-playbook`

完整 `CET4` 详细答案册必须按当前考试配置覆盖写作、听力、阅读 Section A/B/C 和汉译英，并执行前述全部题干/选项翻译、证据、干扰项、写作和翻译成句字段。词汇、句子复杂度、推理深度、录音速度、篇幅与时间应符合当前 `CET4` 配置；语言相对基础不等于可以缩短解析。答案册末尾必须给题源/证据角色说明，并明确模拟材料不是官方真题。

### 六级专项规则 `CET6-specific-playbook`

完整 `CET6` 详细答案册必须按当前考试配置覆盖写作、听力、阅读 Section A/B/C 和汉译英，并执行前述全部题干/选项翻译、证据、干扰项、写作和翻译成句字段。按照当前 `CET6` 配置提高同义替换密度、抽象论证、长难句逻辑、说话人保留态度和推断负荷，但不得凭空增加原题没有的复杂性。答案册末尾必须给题源/证据角色说明，并明确模拟材料不是官方真题。

### 四六级选词填空：Reading Section A

`CET4` 与 `CET6` Section A 的详细解析必须包含：

1. 选项归类表：字母、单词、词性、词形、中文义和常用搭配；名词注明单复数，动词注明原形/时态/分词，并区分形容词、副词及其他形式；
2. 完成填空后的全文、全文翻译和核心词汇；
3. 每个空格的所在句翻译、所需语法形式、`grammar_analysis`、`semantic_judgment`、搭配和语篇逻辑；
4. 与其他语法上可能成立的候选词对比，说明其在语义或上下文中为何不成立；
5. 最终答案和一条可迁移的判断规则。

### 四六级段落匹配：Reading Section B

`CET4` 与 `CET6` Section B 必须提供各段全文翻译。每道陈述都要给中文翻译、关键词、匹配段落与准确句子、证据翻译、同义替换和推理。还要说明相邻段落或话题相近段落为什么只是部分匹配或改变了原命题，不能仅凭共同关键词作答。

### 四六级仔细阅读：Reading Section C

`CET4` 与 `CET6` Section C 必须提供全文翻译和核心词汇。每题翻译题干及 A、B、C、D 全部选项；给出段落/句子定位、证据范围、证据翻译、推理、同义替换，并分别排除所有错误选项。

### 考研英语专项规则 `POSTGRADUATE_ENGLISH-specific-playbook`

对 `POSTGRADUATE_ENGLISH`：

- 完形填空（Use of English）必须给出所有选项的中文义和相关词性、完成后的全文与全文翻译、核心词汇，以及逐空语法、搭配、语义、指代和语篇逻辑分析。
- 阅读 `Reading Part A` 必须给出全文翻译、核心词汇、题干与全部选项翻译、准确定位、证据范围与翻译、长难句拆解、推理和所有干扰项排除。
- 新题型 `Reading Part B` 必须翻译文章以及选项库中的每个标题、句子或段落；说明篇章结构信号、定位、同义替换，以及相邻选项为何衔接不上。
- `Reading/Translation Part C`（阅读翻译）必须把每个画线句拆成分句，理清指代和逻辑，推敲关键词，先建立直译骨架，再形成自然、逐句对齐的译文。

### 专四 TEM-4 专项规则 `TEM4-specific-playbook`

`TEM4` 在执行共用详细标准的基础上，还必须满足以下规则。时间、字数及题型是否出现以当前考试配置为准；目标试卷没有的题型不得凭空添加。

- `TEM4 dictation`：限时作答后给出完整原稿和全文翻译，按意群编号，并标出标点、大小写、功能词、词形、连读、弱读和拼写陷阱。每个漏写片段都要区分听辨、切分、语法、拼写、记忆与标点错误，并给精确的“重听—书写”任务。
- `TEM4 listening`：每段对话或短文都给完整原稿与译文；翻译每道题干和全部选项；说明说话人身份、态度、话轮、转折、数字、因果及推断；给出准确证据行并逐项排除干扰项。
- `TEM4 language knowledge`：翻译题干和所有选项，点明语法、用法、搭配或词义辨析，拆解触发结构，逐项解释，说明语域或例外边界，并给最小对比例句。
- `TEM4 cloze`：提供含中文义和词性的选项表、完成后的全文、全文翻译与核心词汇；逐空分析语法、搭配、语义、指代和篇章逻辑，并比较所有貌似可行的候选词。
- `TEM4 reading`：提供全文翻译和核心词汇；翻译每道题干和所有选项；给出定位、证据范围与译文、同义替换或推断链、决定性长难句拆解，并逐项排除干扰项。
- `TEM4 writing`：完整执行写作契约，包括文体与题目分析、至少三个角度、段落提纲、原创范文和对齐译文、常用及高级话题词汇、可替换槽位模板、量表估分与改写任务。

### 专八 TEM-8 专项规则 `TEM8-specific-playbook`

`TEM8` 在执行共用详细标准的基础上，还必须满足以下规则。时间、字数及题型是否出现以当前考试配置为准；配置标为可选的题型只有实际出现时才讲解。

- `TEM8 mini-lecture`：给出完整讲座原稿与全文翻译，还原提纲层级和篇章信号；翻译每个笔记填空或问题提示；逐空定位答案行，说明所需语法形式和同义替换，并区分内容听辨、笔记记录、拼写与形式错误。
- `TEM8 interview listening`：给出完整访谈原稿与译文；翻译每道题干和所有选项；跟踪说话人、态度、赞同/保留、跨话轮推断和证据范围，并依据原稿逐项排除干扰项。
- `TEM8 reading`：给出全文翻译和核心词汇；翻译题干和所有选项；说明段落/句子定位、证据范围、证据译文、作者态度、篇章结构、长难句逻辑和每个干扰项的错误。主观回答还要列内容得分点和可接受表达。
- `TEM8 language knowledge` 如实际出现，必须执行高级语言知识的全选项翻译、规则触发点、结构拆解、选项对比、语域边界和迁移例句契约。
- `TEM8 bidirectional translation`：英译汉和汉译英都必须给句意分析、分句结构、指代与逻辑、关键词推敲、直译骨架、重组选择、润色后的对齐译文、替代表达、评分点和常见错误修正。英译汉重点解释长难句逻辑与自然中文；汉译英重点解释准确搭配、语域、衔接和地道重组。
- `TEM8 proofreading`：先给改正后的完整文章与全文翻译，再用逐行表格列原错误片段、改正形式、错误类别、控制规则、语境理由和可接受替代。按实际题目覆盖冠词、数、主谓一致、时态、非谓语、指代、连词、介词、搭配、赘余和篇章逻辑，不能只解释替换词。
- `TEM8 writing`：按当前配置的高级要求执行完整写作契约。材料型任务要区分材料概括与个人立场，分析给定观点或数据，提供至少三个可论证角度，并给原创范文与对齐译文、高级话题表达、可替换槽位模板、量表估分与改写任务。不得编造事实或材料观点。

### 完形 `cloze`

- 给出完成后的全文翻译和核心词汇。
- 每题先翻译全部选项或给出其当前语境义。
- 判断空格所需词性和语法形式，检查搭配、语义、指代与跨句逻辑，并比较所有貌似可行的选项。
- 逐题讲解后总结全文叙事或论证逻辑。

### 语言知识 `language-knowledge`

- 翻译有助于理解时，题干和所有选项都要翻译。
- 点明所考语法、用法、搭配或词义辨析，指出句中触发点并简要画出结构。
- 说明例外、语域限制并逐项比较。
- 给一个最小对比例句或改正例句用于迁移。

### 翻译 `translation`

每道翻译必须包含：

1. `sentence_intent_analysis`：说明每个原句表达什么，以及它与前后句的关系；
2. `clause_map`：分析主语、谓语、宾语/补语、修饰语、嵌套从句、逻辑、时态、语态、指代和连接词；
3. `keyword_deliberation`：关键表达有选择空间时列出两种或以上译法，比较意义、搭配和语域，并说明最终选择为何更好；
4. `translation_build`：展示分句直译骨架、必要的结构调整和最终润色句；
5. 原创 `reference_answer`，与原文逐句对齐，并列出可接受替代表达；
6. 从信息、语法、选词、衔接和句式变化说明评分点，再给常见误译与修正任务。

不得只列参考译文。必须说明它为什么准确、自然，也不得把 `R` 级教材或语料库描述成官方固定翻译题库。

### 写作 `writing`

每道写作必须包含：

1. `task_analysis`：判断是议论文、报告、邮件、书信、通知、建议书、图表/图画作文或其他文体，并说明读者、目的、话题、必写要点、立场、字数和评分重点；
2. `idea_bank`：至少三个可选写作角度，每个角度包括观点、理由、例子或结果，以及是否适合本题；
3. `answer_plan`：范文之前先列段落功能、主题句、论证逻辑、例子、衔接和结论；
4. 原创 `reference_answer` 以及逐句或逐段对齐的 `reference_translation`；
5. `topic_vocabulary`：话题常用词和高级词/短语，附中文义、搭配、语域说明和简短可用例句；
6. `reusable_templates`：按功能提供可迁移句型或段落框架，用 `[TOPIC]`、`[REASON]`、`[EXAMPLE]`、`[ACTION]` 等清楚标出可替换槽位；说明适用位置、边界，并至少展示一次替换为其他话题后的用法；
7. 范文逐段点评、衔接与语法说明、可接受的其他论点、量表估分、常见失误和学习者改写清单。

模板用于组织思路，不是万能句。不得鼓励背诵与题目无关的内容、编造数据，或使用与任务语域冲突的模板。

`POSTGRADUATE_ENGLISH` `Writing Part A` 要区分邮件、书信、通知、备忘录等应用文，明确称呼、目的、必写点、语气和结尾。Part B 要先分析图画、图表、文字说明或命题，再分开处理描述、寓意/趋势、原因或影响以及回应。两部分仍必须提供范文译文、话题词汇和可替换槽位模板。

### 改错 `proofreading`

- 给出原形式、改正形式、整句中文义、错误范围和错误类别。
- 在完整句子中解释控制该处的语法、搭配、指代或语篇规则，不能只讲被改的单词。
- 说明其他改法是否成立，比较易混形式，并给一个平行例句。
- 映射到相应改错或语言准确性标签。

### 听写 `dictation`

- 作答后提供正确录音稿和译文，按意群划分并保留标点与句界。
- 对每个漏听片段解释连读、弱读、重音、同音词、功能词、词形、拼写线索和句意。
- 区分听辨、拼写、语法、标点或记忆失败，并分配相应错误标签。
- 给出“重听—分块—写下—核对”的操作序列，而不是要求死记全文。

## 完整试卷答案册顺序

完整模拟卷按以下顺序呈现：

1. 用于快速核对的答案表；
2. 写作审题、观点库、提纲、范文、范文译文、词汇、可替换模板、点评与量表估分；
3. 完整听力原稿和译文，再给逐题题干/选项翻译、证据、推理与干扰项；
4. 完成后的阅读/完形文章、全文翻译、核心词汇和分题型详解；
5. 翻译句意分析、分句结构、关键词推敲、成句过程、对照译文和评分点；
6. 该考试要求的改错、听写或语言知识解析；
7. 错误标签汇总和按优先级排列的重做计划；
8. 题源与证据角色说明，并明确该材料不是官方真题。

答案、解析、录音稿、文章和题源的题号必须一致。不得为了缩短答案册而把多道题合并成一段泛泛说明。

## 交付前质量门

交付前逐项核验：

- 每个题号恰好有一个 `answer_key` 和一份独立解析；
- 每道客观题均有 `question_translation`、所有选项译文、真实 `evidence_anchor`、`evidence_scope`、`evidence_translation`、推理、同义替换和逐个错误选项分析；
- 完整详细答案册中的每篇阅读和听力原稿均按要求配有全文中文翻译与核心词汇；
- 四六级 Section A 有选项归类、完成后的全文、全文翻译，以及逐空语法分析和语义判断；
- 四六级 Section B 有题干翻译、准确段落/句子定位、同义替换和相邻段落排除；
- 四六级 Section C 与考研阅读 Part A 有题干/全选项翻译、证据位置与范围和逐项排除；
- 四级和六级答案分别执行具名专项 playbook、当前配置难度校准、完整分区解析包和非官方声明；
- 专四答案按当前配置包含听写、听力、语言知识、完形、阅读和写作专项解析包；
- 专八答案按当前配置包含讲座/访谈听力、阅读、双向翻译、改错和写作专项解析包；
- 每道写作都有文体/读者/目的分析、三个以上可用角度、提纲、原创范文和译文、常用及高级话题词汇和可替换槽位模板；
- 每道翻译都有句意分析、分句结构、关键词推敲、成句步骤、对照参考译文、替代表达和评分点；
- 答案、证据、同义替换、译文与解析彼此一致；
- 不得编造引文、标题、页码、时间点、规则、证据等级或发布日期；
- 范文与模拟题必须原创或合规改编；
- 分数须标为训练参考，并明确材料不是官方真题或官方固定答案库。

任一项不合格，都应先修复解析或题目再交付。不得以学习者基础水平为由省略必填部分。
