# 考试配置

Skill 支持三条考试路线，以及一组基础水平和目标区间。

## 考试类型

| 值 | 用途 |
| --- | --- |
| `CET4` | 大学英语四级备考。 |
| `CET6` | 大学英语六级备考。 |
| `POSTGRADUATE_ENGLISH` | 中国研究生入学考试英语备考。 |

## 基础水平

| 水平 | 操作含义 |
| --- | --- |
| `基础偏弱` | 需要高频词汇、语法修复、慢速阅读支持和引导式输出。 |
| `中等基础` | 能完成常规任务，但需要提升准确率、速度和考试策略。 |
| `基础较好` | 需要高级准确性、时间控制、表达质量和弱点打磨。 |

校验现有模板时，应使用 `validate_profile.py` 期望的精确本地值。如果终端显示乱码，不要随意改字节；请用支持 UTF-8 的编辑器查看后再修改标签。

## 目标区间

| 考试类型 | 支持目标区间 |
| --- | --- |
| `CET4` | `425~499`、`500~550`、`550+`、`600+` |
| `CET6` | `425~499`、`500~550`、`550+`、`600+` |
| `POSTGRADUATE_ENGLISH` | `50+`、`70~80`、`80+`、`90+` |

## 规划建议

- `基础偏弱`：偏向短任务、错误修复和高频复习。
- `中等基础`：平衡新练习、定向纠错和限时训练。
- `基础较好`：优先处理高价值重复错误、写作/翻译质量和考试时间控制。
- 目标区间会影响训练强度和输出质量预期，但每日计划仍必须遵守 `daily_time_budget_minutes`。
