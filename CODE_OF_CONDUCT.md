# Code of Conduct

## Expected Behavior

Be respectful, specific, and constructive. This project is for exam preparation tooling, so feedback should focus on learner outcomes, correctness, privacy, and maintainability.

## Unacceptable Behavior

Unacceptable behavior includes harassment, personal attacks, sharing private learner data, publishing private prompt bodies, or pressuring maintainers to bypass validation and safety rules.

## Enforcement

Maintainers may edit, hide, or remove comments and contributions that violate this code of conduct. Repeated or severe violations may lead to blocked participation.

## Reporting

Report conduct issues to the project maintainers. Do not include private learner records or private prompt text in public reports.
