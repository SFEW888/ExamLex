# Roadmap

The roadmap stays intentionally short so it reflects maintainable work.

## Done

- [x] Public-safe Skill package for CET-4, CET-6, and postgraduate English.
- [x] CLI for profile validation, daily planning, practice logging, error summaries, ability updates, trends, and writing scoring.
- [x] Platform adapter notes for Claude Code, Codex CLI, Codex App, and Cursor.
- [x] Repository validation, unit tests, CI, issue templates, and prompt-safety checks.

## Planned

- [ ] Add richer example workflows for weak, middle, and strong foundation learners.
- [ ] Add optional coverage reporting in CI.
- [ ] Add a local-only private prompt loading example that never enters public-safe artifacts.
- [ ] Add release automation after the first stable public contract.

## Not Planned For The First Public Version

- Hosted learner data storage.
- Cloud account management.
- Guaranteed official exam score prediction.
