# Postgraduate English Guide

Postgraduate English preparation should combine vocabulary depth, reading inference, translation precision, writing structure, and long-term error review.

Supported target bands:

- `50+`: build stable foundations and reduce avoidable grammar or vocabulary errors.
- `70~80`: strengthen reading accuracy, translation choices, and structured writing.
- `80+`: focus on high-yield repeated errors, timing, and quality of expression.
- `90+`: emphasize precision, consistency, advanced wording, and full-paper strategy.

Example Agent calls:

```text
/structure-planner Build an outline for this postgraduate English essay prompt.
/polish-wizard Improve this draft while preserving my meaning.
```

The rubric estimate helps compare drafts and guide revision. It is not an official postgraduate entrance exam score guarantee.
